# S2S3 开荒队阵容与 T 度参考

---

## 开荒 T0 — 苦肉弓

- 阵型：箕形阵
- 黄盖：技能 坚如磐石、屈人之兵 / 下位战法 坚壁清野 / 加点 至少30速，其余全统
- 孙权：技能 青囊急救、【料事如神】 / 下位战法 清风/东手 / 加点 比黄盖快一速，其余全智
- 周瑜：技能 【胜敌益强】（优先）、【S2奇正相生 / S3明其虚实】 / 下位战法 王佐/出其 / 加点 全智
