# 战法替换参考

<!-- 触发关键词：战法替换 下替战法 战法下位 战法替代 减伤替代链 底层逻辑 -->

> 推荐战法为标准配置，下替战法为无法获取时的替代选项。部分替换有方向性说明，请按需选择。

---

## 底层逻辑

队伍四维之间可相互转换：**增伤、出伤、减伤、控制**。

当减伤实在不够用时，参考下方减伤替代链进行替换。

---

## 减伤替代链

按优先级从高到低，依次降级使用：

**遇其锐气** → **横征暴敛**（有条件） → **以静制动** → **坚如磐石** → **如沐春风** → **来好息师**（双前排）

## 主动战法替换

### 指点乾坤
- 控制链方向 → 巧利天灾 + 狂风大作
- 单控方向 → 料事如神 / 水淹七军
- 增伤方向 → 战八方

### 万夫莫当
- 下替 → 强袭

### 料事如神
- 下替 → 束手无策

### 王佐之才
- 下替 → 出其不意

### 兵贵神速 / 铁骑横冲
- 下替 → 横扫千军 → 千里突袭 → 夜袭

### 战八方
- 下替 → 疾行侧击

### 水淹七军
- 主C位 → 换出伤战法
- 辅助位 → 换攻其不备 / 料事如神

---

## 被动战法替换

### 轻装驰援
- 下替 → 铸甲销戈

### 披坚执锐
- 下替 → 遇其锐气 / 以静制动 / 兵书临峰

### 遇其锐气
- 高防方向 → 以静制动 + 铸甲销戈
- 高连击方向 → 高连击率 + 轻装驰援

### 锐不可当
- 主动触发方向 → 势如破竹
- 追击触发方向 → 铁骑横冲

### 无难之志
- 刘备使用 → 搭配方圆阵 + 青囊急救
- 乐进使用 → 任人唯贤
- 袁绍使用 → 青囊急救 / 指点乾坤

### 烈火张天S3
- 下替 → 烈火焚营 → 火羽

---

## 辅助/控场战法替换

### 明其虚实
- 下替 → 出其不意

### 舍生取义
- 阵型配合 → 锥形阵 + 洞若观火
- 无替换时 → 直接不带，换增伤赌脸

### 蓄势待发
- 下替 → 百战不殆 → 上兵伐谋（次级下替）

### 百战不殆
- 下替 → 直接不带，换增伤 / 出伤战法

### 摧坚克难
- 下替 → 势如破竹

### 胜敌益强
- 下替 → 锐不可当

### 韬光养晦
- 下替 → 如有神助
