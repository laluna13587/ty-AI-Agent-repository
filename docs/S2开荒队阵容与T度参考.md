# S2 开荒队阵容与 T 度参考

---

## 20级前首选 — 蔡袁董

> 备注：二十级之前最佳开荒队伍。

- 阵型：箕形阵
- 袁绍：技能 遇其锐气 / 加点 全统
- 董卓：技能 胜敌益强 / 加点 一速全智
- 蔡文姬：技能 指点乾坤 / 加点 全智

---

## 开荒 T0 — 苦肉弓

> 备注：20级之后最佳开荒队伍。没铜币时料事如神最后升级。相比刘法马兵损偏高，非常在意兵损的玩家可以优先考虑刘法马。

- 阵型：箕形阵
- 黄盖：技能 坚如磐石、屈人之兵 / 下位战法 坚壁清野 / 加点 全速
- 孙权：技能 青囊急救、【料事如神】 / 下位战法 清风/束手 / 加点 全速
- 周瑜：技能 【胜敌益强】（优先）、奇正相生 / 下位战法 王佐/出其 / 加点 全智

---

## 开荒 T0.3 — 小黄鱼

- 阵型：箕形阵
- 黄盖：技能 坚如磐石、屈人之兵 / 下位战法 坚壁清野 / 加点 至少30速，其余全统
- 小乔：技能 青囊急救、【料事如神】 / 下位战法 清风/束手 / 加点 比黄盖快30速，其余全智
- 周瑜：技能 【胜敌益强】（优先）、奇正相生 / 下位战法 王佐/出其 / 加点 全智

---

## 开荒 T0.5 — 刘法马 / 神威

> 备注：最佳二队，同红度开地比苦肉弓晚两级。兵损低于苦肉弓，在意兵损的玩家推荐优先考虑本队。

- 阵型：箕形阵
- 刘备：技能 遇其锐气、披坚执锐 / 下位战法 铸甲销戈 / 加点 全智
- 法正：技能 指点乾坤、束手无策（打主动）/屈人之兵（打普攻） / 下位战法 束手/披坚 / 加点 一速全智
- 马超：技能 势如破竹、【擒龙克难】（优先） / 加点 15速，其余全武

---

## 开荒备选 — 貂蔡张

> 备注：前面三队（蔡袁董/苦肉弓/刘法马）凑不齐时可用，强度比刘法马略低。开服两小时后领话本张梁。

- 阵型：箕形阵
- 貂蝉：技能 披坚执锐、青囊急救 / 加点 全统
- 蔡文姬：技能 料事如神、指点乾坤 / 加点 20速全智
- 张梁：技能 胜敌益强、锐不可当 / 加点 全武

---

## 开荒 T0.8 — 桃园 / 变种桃

- 阵型：箕形阵
- 刘备：技能 【遇其锐气】（优先）、青囊急救 / 下位武将 曹操/月英 / 下位战法 清风/披坚 / 加点 全智
- 张飞：技能 势如破竹、【威名显赫】（优先） / 下位战法 疾行/绑甲/伏兵 / 加点 比关羽慢1速
- 关羽：技能 战八方、水淹七军 / 下位战法 斩将/势如/威名 / 加点 比张飞快1速，至少15速，其余全武

## 开荒 T1 — 孙太大

- 阵型：箕形阵
- 大乔：技能 遇其锐气、无难/青囊急救 / 下位战法 清风/披坚 / 加点 全智
- 孙策：技能 战八方、水淹七军/势如破竹 / 下位战法 疾行/绑甲/伏兵 / 加点 15速，其余全武
- 太史慈：技能 兵贵/攻其/横扫、轻装驰援 / 下位战法 斩将/势如/威名 / 加点 25速，其余全武

## 35级后更强 — 天灾开荒

> 备注：优先度比较靠后，除非没有高强度的队伍可玩，否则不太推荐优先出这队。

- 阵型：箕形阵
- 曹操：技能 遇其锐气、青囊急救 / 下位战法 清风驱疾 / 加点 全智
- 贾诩：技能 烈火焚营、狂风大作、巧利天灾 / 下位战法 束手 / 加点 全智
- 周瑜：技能 胜敌益强、奇正相生 / 下位战法 有、出其 / 加点 全智
