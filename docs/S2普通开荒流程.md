**白板缺武将战法使用**

> 参考攻略来源：2025.10.27 雕馄 & 柳絮修订

- 慢就是稳，稳就是快，别越级开也能上榜

## 第一天

**步骤1** 落地直接用五子/任意杂牌队开垦4块1级地，目的是节省主力队体力。
- 队伍：杂牌五子/任意一队，箕形阵
- 配置：字幕（全统）、张部（全武）、涂昱（全武）

**步骤2** 下阵，上阵孙太大，开垦4块2级地，升到8级。

**步骤3** 8级孙太大打2块外城4级地（刘晔/沙摩柯），升到10级。翻车总有概率，拿到经验就好。

队伍：平民孙太大 · 箕形阵
- 大乔：战法 披坚执锐（下替遇其）/ 加点 全智
- 孙策：战法 战八方（下替势如、威名）/ 加点 +20速其余全武
- 太史慈：战法 横扫千军（下替兵贵、攻其）/ 加点 速度第二其余全武

**步骤4** 下阵，上阵打工队，开垦内城剩余8块3级地，然后下阵。

**步骤5** 上阵孙太大，装上10级新技能。

**步骤6**
- 10级孙太大再打几块任意4级地，升到11级
- 11级打2块外城5级地（三紫沙摩柯）
- 12级起无脑刷5级地到20级

**步骤7** 20级换队优先级：小黄鱼 > 刘法马 > 桃园；都没有则继续孙太大，打6级地刘晔。

队伍：小黄鱼 · 箕形阵
- 黄盖：战法 坚如磐石（下替固若金汤、坚壁清野）、屈人之兵 / 加点 全速 / 兵书 金书、机动、励心
- 小乔：战法 青囊急救、料事如神（下替束手）/ 加点 全速 / 兵书 公瑾信、机动、励心
- 周瑜：战法 胜敌益强、奇正相生（下替出其不意）/ 加点 全智 / 兵书 三增伤

**步骤8** 屯1块6级石头，过任务。

**步骤9** 白板25级开7级地（优先程普/甄姬）。

**步骤10** 屯1块7铁、1块7木、2块7石，解锁兵营，瞬间升3/4级兵营。

**步骤11** 刷7级地，避开黄忠/张宝/张梁。

> 关于挂机：4小时后解锁内城挂机功能，上阵3个5级武将，挂机2小时到12级，开始开垦内城4级地→5级，开垦完继续挂机。

**步骤12** 白板30级开8级地（程普/张飞/许褚）。

**步骤13** 刷8级地。第一天结束，非全图鉴白板目标：**30级达标**。

---

## 第二天

**步骤14** 刷8级地到35级，洗兵书，打9级地（邓艾/张飞）。

**步骤15** 优先打城外9级地，出生州9级地数量有限，一般第二三天会开始卷。

> 关于二队：前期可以把一队电表倒转当二队用（一队换成3紫将再换去二队），只用一队当两队打。懒得来回换就直接用刘法马。

队伍：二队-刘法马 · 箕形阵
- 刘备：战法 遇其锐气、披坚执锐 / 加点 全智
- 法正：战法 共用料事、指点乾坤 / 加点 速度第一其余全智
- 马超：战法 擒坚克难、势如破竹 / 加点 速度第二其余全武

**步骤16** 白板40级打10级地。

---

## 第三天

**步骤17** 白板43级打11级地。

**步骤18** 白板45级打12级地。

> 远征极其重要，远征打得好多几百资源箱。远征队优先级：**远征苦肉弓 > 小黄鱼**

队伍：远征版小黄鱼 · 箕形阵
- 黄盖：战法 坚如磐石、屈人之兵 / 加点 全速 / 兵书 金书、机动、励心
- 小乔：战法 青囊急救、一计决胜 / 加点 全速 / 兵书 公瑾信、机动、励心
- 周瑜：战法 胜敌益强、奇正相生 / 加点 全智 / 兵书 三增伤

**步骤24** 大概下午12点-6点，39/40级以上用小黄鱼打远征（不能40级就39打，不要太晚，尽快拿资源）。

远征增益优先级：1规避 → 2发动 → 3增伤 → 4减伤 → 5智力 → 6统率
